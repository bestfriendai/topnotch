Privacy Policy

Last updated: March 20, 2026

Overview

Top Notch is designed with privacy as a core principle. The app runs entirely on your Mac and does not collect, transmit, or store any personal data on external servers.

Data We Do Not Collect

Top Notch does not collect:

  • Personal information (name, email, phone number)
  • Usage analytics or telemetry
  • Crash reports sent to third parties
  • Advertising identifiers
  • Location data

Data Stored Locally

The following data is stored locally on your Mac and never leaves your device:

  • Clipboard History: Your last 20 copied items are stored in app preferences for the Clipboard widget. You can clear this at any time in Settings.
  • App Preferences: Widget settings, accent color, weather city, and display preferences are stored in macOS UserDefaults.
  • YouTube History: Recently played YouTube video IDs (up to 5) are stored locally for quick replay.
  • Quick Notes: Notes created in the Quick Capture widget are stored locally in app preferences.
  • API Keys: If you configure an optional Claude API key for AI features, it is stored securely in the macOS Keychain, not in plaintext.

Network Requests

Top Notch makes network requests only for the following features:

  • Weather: City name and coordinates are sent to the Open-Meteo API (open-meteo.com) to fetch weather data. No personal data is included.
  • Lyrics: Song title and artist name are sent to LRCLIB (lrclib.net) to fetch synced lyrics. No personal data is included.
  • YouTube: YouTube video playback uses an embedded web view that connects to YouTube's servers. YouTube's own privacy policy applies to video playback.
  • AI Summarization (optional): If you configure a Claude API key, clipboard text may be sent to the Anthropic API for summarization. This feature is disabled by default and requires explicit opt-in.

Calendar Access

Top Notch requests access to your calendar via Apple's EventKit framework to display today's events in the Calendar widget. Calendar data is read locally and is never transmitted over the network. You can deny calendar access and the widget will show a prompt to enable it in System Settings.

Apple Events

Top Notch uses Apple Events (AppleScript) to communicate with Apple Music and Spotify for media playback control. This is used solely to play, pause, skip tracks, and retrieve now playing information.

Third-Party Services

Top Notch does not include any third-party analytics, advertising, or tracking SDKs.

Children's Privacy

Top Notch does not knowingly collect any information from children under 13.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated revision date.
